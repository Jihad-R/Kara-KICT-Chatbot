<h3>Installation</h3>
See Getting Started to install requirement tools.
<ul>
<li>$ npm install expo-Speach </li>
<li>$ npm install react-native-gifted-chat --save</li>
</ul>
<h3>Development</h3>
<ul>
<li>
Start local server
</li>

iOS
